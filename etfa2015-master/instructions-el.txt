Προαπαιτούμενα:
1.Eγκατεστημένη Python 2.7.3

2.Eγκατάσταση βιβλιοθήκης modbus-tk-0.4.2
  α) Αποσυμπίεση modbus-tk-0.4.2.zip
  b) python setup.py install

3.Eγκατάσταση βιβλιοθήκης scapy 2.2.0

4.Μέσα από το φάκελλο /root/mtf, εκτελούμε το σενάριο  mtf.py με παραμέτρους όπως παρακάτω:

	python mtf.py -i <host>  -s <search_mode> -z <fuzz_mode> -f <csvFile=search.csv> -p <pcap_file=packets.pcap> -r <fuzz_reguest>'   

	host            - the IP of the remote machine (under test)
	csvFile         - stores search information
	pcap_file       - trace pcap file 
	fuzz_reguest, αριθμός ερωτημάτων για κάθε FC που εντοπίζεται απο την αναγνώριση (προεπιλογή=200)

Ι. python mtf.py -i 192.168.Χ.Χ -s -f, διαδικασία της αναγνώρισης, όπου η 192.168.Χ.Χ είναι η Ιp συσκευής ή λογισμικό με open port 502 
Αποτέλεσμα α)παραγωγή αρχείου με ονομασία search.csv. Το αρχείο περιέχει  λίστα υποστηριζόμενων FC και διευθύνσεων της συσκεύης ή λογισμικού που δοκιμάζεται
β)παραγωγή αρχείου με ονομασία dump_memory.csv, αποθήκευση πληροφορίας της επίθεσης dump memory attack (ανάκτηση πληροφοριών σχετικά περιεχόμενα των θέσεων μνήμης της remote machine (SUT)
(Address 0x Value READ_COILS", "Address 3x Value READ_INPUT_REGISTERS " κ.λ.π) γ)παραγωγή log αρχείων της μορφής info_Y_m_d_H_M_S.log (Ετος_Μήνας_ημέρα_Ωρα_λεπτά_δευτερόλεπτα.log) και 'error_Y_m_d_H_M_S.log' με πληροφορίες συναλλαγών κατά τη διαδικασία της αναγνώρισης.
 
ΙΙ. python mtf.py -i 192.168.Χ.Χ -s -z -f  διαδικασία της αναγνώρισης (φάση  1)  και στην συνέχεια fuzzing (φάση 2).
Αποτέλεσμα α)παραγωγή αρχείου info_Y_m_d_H_M_S.log και 'error_Y_m_d_H_M_S.log' με πληροφορίες συναλλαγών (ερωτήματα-απαντήσεις) και λάθη. Απο τα παραπάνω παίρνουμε πληροφορία για να εξάγουμε συμπεράσματα
παράγονται επίσης τα α) και β) του (Ι)

ΙΙΙ. python mtf.py -i 192.168.Χ.Χ  -z -f διαδικασία fuzzing (φάση 2), προυπόθεση η ύπαρξη αρχείου search.csv (παραγέται απο τη φάση 1) (Ι) 

IV python mtf.py -i 192.168.Χ.Χ -s -f -p, διαδικασία της αναγνώρισης απο αρχείο κίνησης pcap (packets.pcap). Παραγωγή αρχείου με ονομασία search.csv όπως στο  Ι. Το αρχείο περιέχει  λίστα υποστηριζόμενων FC και διεθύνσεων βάσει της κίνησης του αναγνώστηκε απο το δωθέν αρχείο pcap
   (δοθήσης και της παράμετρου -z, τότε ο MTF αρχίζει αυτόματα και διαδικασία fuzzing)
